## 🤖 API Checker AI

**API Checker AI** is a lightweight tool designed to validate, test, and analyze API endpoints using AI-powered insights.

It helps developers quickly identify issues, verify responses, and understand API behavior without manually digging through logs or writing repetitive test cases.

### ✨ Features

* 🔍 **Smart API Validation**
  Automatically checks endpoint responses, status codes, and structure.

* 🧠 **AI-Powered Analysis**
  Understands responses and highlights potential issues or inconsistencies.

* ⚡ **Fast & Lightweight**
  Built to run efficiently, with minimal setup required.

* 🔐 **Header & Request Handling**
  Supports custom headers, body forwarding, and real-world request simulation.

* 🌐 **Proxy-Friendly**
  Can be used as a relay or middleware to inspect live API traffic.

### 🚀 Use Cases

* Debugging API responses in development
* Monitoring third-party APIs
* Testing edge cases and unexpected inputs
* Building smarter API testing workflows

### 🛠️ Getting Started

Clone the repository and deploy or run locally:

```bash
git clone https://github.com/your-username/api-checker-ai.git
cd api-checker-ai
npm install
npm run dev
```

### ⚙️ Configuration

Make sure to set your target API:

```env
TARGET_DOMAIN=https://your-api.com
```
